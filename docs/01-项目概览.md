# 项目概览
## 项目定位
Vue 3 + TypeScript 入门实战项目，帮助初学者理解前后端如何通信。包含一个 Vue 3 前端和两个功能完全等价的后端（TypeScript / Python），在对比中学习。
## 功能一览

| 功能 | 说明 | 涉及知识点 |
|------|------|------------|
| 计数器 + 双向绑定 | 输入名字、加减计数 | ref、computed、v-model、v-if、v-for |
| 本地待办事项 | 纯前端，数据存在内存 | 组件化、响应式数组操作 |
| API 待办事项 | 通过 fetch() 调用后端 CRUD | REST API、JSON、onMounted |
| 后端切换 | 下拉框一键切换 TS / Python 后端 | 同一套 API，不同实现 |
| 实时监控面板 | WebSocket 推送 CPU/内存/请求日志 | WebSocket、断线重连、实时数据流 |

## 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | Vue 3 + TypeScript + Vite | Vue 3.5 / TS 5.7 / Vite 6 |
| TS 后端 | Express + better-sqlite3 + ws | Express 4 |
| Python 后端 | Flask + flask-sock + psutil | Flask 3.1 |
| 数据库 | SQLite（零配置，文件型） | — |

## 项目结构
```plaintext
vue3-starter/
├── index.html                     # 入口 HTML
├── package.json                   # 前端依赖
├── vite.config.ts                 # Vite 配置
├── tsconfig.json                  # TypeScript 配置
├── frontend/                      # 前端源码
│   ├── main.ts                    # 应用入口，挂载 Vue 实例
│   ├── App.vue                    # 根组件（计数器 + 概念列表）
│   └── components/
│       ├── TodoList.vue           # 待办事项（纯前端版）
│       ├── TodoListApi.vue        # 待办事项（API 版）
│       └── Monitor.vue            # 系统监控面板（WebSocket）
├── backend-ts/                    # TypeScript 后端
│   └── src/
│       ├── index.ts               # Express 路由 + WebSocket 服务
│       └── db.ts                  # SQLite 数据库 CRUD 操作
└── backend-py/                    # Python 后端
    ├── app.py                     # Flask 路由 + WebSocket 服务
    └── db.py                      # SQLite 数据库 CRUD 操作

```
