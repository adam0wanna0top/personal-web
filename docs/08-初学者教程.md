# 初学者教程
## 一、项目目录结构
```plaintext
vue3-starter/
├── index.html                  # 网页入口（Vite 以此为起点）
├── package.json                # 项目描述：依赖、脚本命令
├── vite.config.ts              # Vite 构建工具配置
├── tsconfig.json               # TypeScript 主配置
├── tsconfig.node.json          # TypeScript 针对 Node/Vite 的配置
├── node_modules/               # npm 安装的依赖包（不要手动改）
├── frontend/
│   ├── main.ts                 # JS/TS 入口：创建 Vue 应用
│   ├── App.vue                 # 根组件（第一个 Vue 组件）
│   ├── vite-env.d.ts           # 告诉 TS 什么是 .vue 文件
│   └── components/
│       └── TodoList.vue        # 子组件

```

## 二、每个文件是干什么的
### index.html — 网页入口
```html
<div id="app"></div>                          <!-- Vue 挂载点 -->
<script type="module" src="/frontend/main.ts"></script>  <!-- 加载 TS 入口 -->

```

- `<div id="app">` 是一个空容器，Vue 会把整个应用渲染到里面
- `type="module"` 表示用 ES Module 方式加载脚本
- Vite 开发服务器会自动处理 `.ts` 文件
### package.json — 项目描述
```plaintext
{
  "name": "vue3-starter",
  "scripts": {
    "dev": "vite",              // npm run dev → 启动开发服务器
    "build": "vite build",      // npm run build → 打包
    "preview": "vite preview"   // npm run preview → 预览
  },
  "dependencies": {             // 运行时依赖
    "vue": "^3.5.0"
  },
  "devDependencies": {          // 开发时依赖
    "@vitejs/plugin-vue": "^5.2.0",
    "typescript": "^5.7.0",
    "vite": "^6.0.0"
  }
}

```

### vite.config.ts — 构建工具配置
```typescript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],   // 注册 Vue 插件
})

```

Vite：超快的开发服务器 + 构建工具。开发时按需编译，浏览器瞬间热更新。
### tsconfig.json — TypeScript 配置
关键配置：
- `"strict": true` — 开启所有严格检查
- `"paths": { "@/*": ["./frontend/*"] }` — 路径别名
### frontend/main.ts — 应用入口
```typescript
import { createApp } from 'vue'
import App from './App.vue'

createApp(App).mount('#app')

```

执行流程：index.html → main.ts → 创建 Vue 应用 → 渲染 App.vue
## 三、.vue 文件（单文件组件 SFC）
每个 .vue 文件由三部分组成：
```plaintext
┌─────────────────────────┐
│  <template>             │  ← HTML 模板（页面结构）
│    <div>...</div>       │
│  </template>            │
├─────────────────────────┤
│  <script setup lang="ts">│ ← TS 逻辑（数据、方法）
│    const count = ref(0) │
│  </script>              │
├─────────────────────────┤
│  <style scoped>         │  ← CSS 样式（scoped = 只作用于此组件）
│    h1 { color: red; }   │
│  </style>               │
└─────────────────────────┘

```

`<script setup>` 是 Vue 3 的语法糖，里面声明的变量和函数自动可在模板中使用。
## 四、Vue 3 核心 API 速查
### 响应式数据
```typescript
import { ref, reactive } from 'vue'

const count = ref(0)         // ref：基本类型，JS 中用 .value 读写
const user = reactive({})    // reactive：对象，直接读写属性

```

### 计算属性
```typescript
const doubled = computed(() => count.value * 2)  // 依赖变化时自动重算

```


## 五、如何启动
```bash
cd vue3-starter
npm install     # 安装依赖（第一次）
npm run dev     # 启动开发服务器

```

打开 http://localhost:5173 即可看到页面。
## 六、开发流程
```plaintext
1. 修改 .vue 文件 → 保存 → 浏览器自动刷新
2. 新建组件 → 在父组件中 import → 使用
3. 出错了 → 看浏览器控制台（F12）

```

新建组件步骤：
1. 在 frontend/components/ 下创建 XxxView.vue
1. 写好 template + script setup + style scoped
1. 在父组件中 import 并使用
